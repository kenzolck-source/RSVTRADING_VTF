尋找最佳狙擊點

先判斷市場有無清晰結構，再決定應該落盤、縮 size，定係離開滑鼠。

Step 1

蠟燭有「尊重」系統嗎？

是否規律貼著 EMA 或在雲層邊緣反彈？

新手看盤提示

  • 先縮小圖表。不要只看最後一支 K，先看最近 15-30 分鐘是否有一致反應。
  • YES 的證據：同一條 EMA、Cloud 邊、前高低被市場反覆尊重，價格到位後有明確反應。
  • NO 的證據：上下亂插、雲內雲外來回穿、每一支 K 都打破上一個判斷。
  • 小白規則：如果你要用很多說話說服自己「其實都有少少尊重」，就當 NO。
Step 2

大方向是誰在控盤？

先決定大局。大腦要猜，就代表暫時看不懂。

新手看盤提示

  • Uptrend：高點有推高，回調低點守得住，價格大多在 Cloud 上方或被買回。
  • Downtrend：低點有跌穿，反彈高點過不到，價格大多在 Cloud 下方或被賣回。
  • Range：上下邊界都明顯，中間沒有方向優勢，只能等低位做多或高位做空。
  • 看不懂：牛熊理由同時成立，或者你 10 秒內講不出主導方，就先 WAIT。
Step 3

大 Time Frame 有阻礙嗎？

小級別 setup 不應硬撞 1H / 4H 強雲層或關鍵位。

新手看盤提示

  • 先看目標方向。如果做多，就看上方最近有沒有 1H/4H Cloud、前高、供應區；做空就看下方。
  • 暢通：Entry 到第一個明顯阻力/支撐之間有空間，R:R 有機會達標。
  • 擋路：一入場很快撞大級別位，即使方向看對，也可能只賺少少但要承受完整止損。
  • 小白規則：如果 target 比 stop loss 還近，這不是好交易，是追緊張感。
Step 4

客觀 Setup 偵測

先在「指標快填」輸入你看到的事實：Cloud 狀態、Cloud-to-Cloud 路徑、Trap、VSOP 四步、Kim / R-Kim。然後由系統判斷最適合的 setup；證據不夠就 WAIT。

新手看盤提示

  • 不要先決定 Setup 名稱。先填你看見的證據，再讓系統告訴你最像哪一種。
  • FIRST TRAP：重點是先掃 liquidity 或假突破，再 reclaim 關鍵位。
  • VSOP：四步要齊：Imbalance、FVG、Sweep Liquidity、Back to FVG。
  • CLOUD TO CLOUD：中間路徑要乾淨，下一層 Cloud 才是合理 target。
  • KIM / R-KIM：先有明顯 impulse，再弱反彈或小回調，最後突破 trigger。

由盤面事實決定 Setup

這一步不需要你選 FIRST TRAP / VSOP / CLOUD TO CLOUD / KIM。你只需要如實填寫觀察，程式會把最合理的 setup 推出來。

Range Edge

Setup 發生在 Range 的哪個位置？

Range 可以交易，但只接受邊界。中間位即使有 setup 名稱，也容易變成追價。

新手看盤提示

  • 低多：價格在 Range 底部附近，有 setup 支持反彈，SL 可以放在地板外。
  • 高空：價格在 Range 頂部附近，有 setup 支持回落，SL 可以放在天花板外。
  • 不做：價格在中間、上下空間差不多、或者你不能清楚說出邊界在哪。
  • 小白規則：Range 中間最容易令人覺得「好似有位」，但通常最沒有 edge。
